## 🔧 Arquitectura del Sistema

### 🏗️ **Flujo de Procesamiento**
```
Input: Research Question + Domain
    ↓
ScientificHypothesisAgent (ATLAS)
    ↓ 
ImprovedHybridConfidenceFilter v2.1
    ├── ML Component (R² = 0.960)
    └── Anti-Pseudoscience Patterns (5 categories)
    ↓
Decision: APPROVE/REJECT + Confidence + Reasons
    ↓
EnhancedHypothesisWorkflow
    ├── Statistics & Monitoring  
    └── Batch Processing Support
```

### 🛡️ **Componentes del Filtro**
1. **Machine Learning:** Gradient Boosting entrenado con 66,461 papers retractados
2. **Pattern Detection:** 25+ regex patterns en 5 categorías de pseudociencia
3. **Exponential Penalty:** Penalización por patrones detectados
4. **Decision Engine:** Umbral 0.5 para APPROVE/REJECT
