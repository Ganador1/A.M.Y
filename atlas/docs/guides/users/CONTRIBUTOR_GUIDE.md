# Guía para Contribuidores en AXIOM

Esta guía es para quienes desean contribuir al proyecto AXIOM.

## Cómo Contribuir

- Fork el repositorio.
- Crea branches para features.

## Estándares de Código

- Sigue PEP8 para Python.

## Proceso de Pull Request

- Envía PR con descripciones claras.

## Comunidad

- Únete a discusiones en GitHub.
