# CHANGELOG - MathLab

## [0.2.0] - 2024-07-19

### Añadido
- Enumeración no isomorfa de grafos usando pynauty
- Invariantes avanzados para grafos (número cromático, clique, independencia, diámetro, radio)
- Embedding basado en histograma de grados para grafos
- Plugin de conjeturas para propiedades cromáticas de grafos

### Mejorado
- Manejo de errores parciales en cálculos por lotes (batch)
- Documentación del progreso en el documento principal del Agente 2

## [0.1.0] - 2024-07-12

### Añadido
- Registro de objetos matemáticos con hashing semántico
- Sistema de invariantes básicos para grafos y números
- Infraestructura de testing
