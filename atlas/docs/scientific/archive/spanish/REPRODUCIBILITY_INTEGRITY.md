# Reproducibilidad e Integridad de Artefactos (Spanish Original)

Este documento describe el flujo completo para asegurar reproducibilidad e integridad criptográfica de modelos y publicaciones en AXIOM META 4.

## Objetivos

- Trazar cada artefacto a su código (commit), dataset, parámetros y métricas.
- Validar estructura mediante JSON Schema (gates en CI).
- Garantizar integridad (hash SHA-256 + firmas Ed25519).
- Facilitar verificación automática (script + job CI) y manual (CLI).
