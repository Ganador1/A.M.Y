# Services in Neuroscience Domain

## AdvancedNeuroimagingService
Description: Advanced processing of neuroimaging data.
Key Features: Multi-modal analysis, real-time processing.
Usage: Use /neuroscience/advanced/analyze endpoint.

## NeuroSimulationService
Description: Simulation of neural systems.
Key Features: Brian2 and NEURON integration.
Usage: POST to /neuroscience/simulation/brian2.

## NeuralNetworksService
Description: Neural network models for neuroscience.
Key Features: CNN, RNN, Transformer support.
Usage: Integrate with ML workflows.

## MultiScaleModelingService
Description: Multi-scale brain modeling.
Key Features: From molecular to system level.
Usage: Define scales and couplings.
