# Neuroscience Domain

## Overview
The Neuroscience domain offers tools for neuroimaging analysis, neural simulations, multi-scale modeling, and neural network processing.

## Available Services
- **NeuroscienceLightService**: Lightweight EEG analysis.
- **AdvancedNeuroimagingAnalysis**: Advanced neuroimaging processing.
- **NeuralNetworksService**: Neural network architectures.
- **NeuroSimulationService**: Neural system simulations.
- **MultiScaleModelingService**: Multi-scale brain modeling.
